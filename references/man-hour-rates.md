## 4. Productivity Factors

These factors are applied as multipliers to base man-hour values.

| Condition | Factor | Description |
|-----------|--------|-------------|
| Normal site (onshore, temperate climate) | 1.00 | Reference value |
| Offshore platform | 1.30 - 1.50 | Limited space, logistics |
| Desert environment (Turkmenistan interior) | 1.15 - 1.25 | Heat, sandstorm |
| Caspian coast | 1.10 - 1.20 | Wind, humidity |
| Work at height (>5m) | 1.15 - 1.30 | Scaffolding requirement |
| Confined space | 1.25 - 1.50 | Safety procedures |
| Ex-zone (Zone 1, Zone 2) | 1.15 - 1.30 | Special equipment/procedures |
| Night shift | 1.15 - 1.25 | Lighting, reduced productivity |
| Winter conditions (<0 deg C) | 1.20 - 1.40 | Warming breaks, clothing |
| Brownfield (working in existing facility) | 1.20 - 1.40 | Limited access, permits |
